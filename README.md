# Buyzaar - Full-Stack E-commerce Website

Buyzaar is a full-stack e-commerce web application built using React.js, Node.js, Express, and MongoDB.  
It allows users to browse products, add items to cart, and place orders.  
Admins can manage products, orders, and images via Cloudinary.

---

## Features

### User
- User registration and login
- Browse products by category
- Add products to cart
- Place orders
- View order history

### Admin
- Add / update / delete products
- Manage orders
- Upload images via Cloudinary
- Stock management

---

## Tech Stack

**Frontend:** React.js, React, Tailwind CSS, Redux Toolkit  
**Backend:** Node.js, Express.js, MongoDB, JWT Authentication, Cloudinary
